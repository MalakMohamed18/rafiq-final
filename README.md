#  تطبيق رفيق - Rafiq Health App

**رفيق** هو تطبيق موبايل متطور مخصص لمراقبة الحالة الصحية للمرضى، يجمع بين واجهة مستخدم عصرية وأدوات تتبع لحظية للمؤشرات الحيوية مع نظام تنبيهات ذكي.

---

## المميزات الرئيسية (Key Features)

* ** تخصيص المستخدم:** عرض اسم المريض الحقيقي وصورته من قاعدة البيانات بدلاً من النصوص الثابتة.
* ** تتبع نبضات القلب (Dynamic BPM):** رسم بياني تفاعلي (ECG) يعرض معدل ضربات القلب الحالي ويغير لونه عند وجود خطر.
* ** تنبيهات لحظية (Real-time Notifs):** نظام إشعارات فوري يعمل بتقنية WebSockets لتنبيه المستخدم بأي تحديثات فور وقوعها.
* ** نظام الثيمات (Theming):** دعم كامل للوضع الداكن (Dark Mode) والوضع الفاتح (Light Mode).
* **🇸🇦 دعم RTL:** واجهة مستخدم مصممة بالكامل لتدعم اللغة العربية من اليمين إلى اليسار.
* ** نظام الطوارئ:** وصول سريع لخدمات الطوارئ (SOS) بضغطة واحدة.

---

##  المكتبات المستخدمة (Dependencies)

تم بناء التطبيق باستخدام أحدث التقنيات لضمان السرعة والأمان:

| المكتبة | الوظيفة |
| :--- | :--- |
| **React Native (Expo)** | الإطار الأساسي لبناء التطبيق لنظامي Android و iOS. |
| **TypeScript** | لضمان جودة الكود وتقليل الأخطاء البرمجية. |
| **@supabase/supabase-js** | لإدارة عمليات تسجيل الدخول وقاعدة البيانات اللحظية. |
| **@expo/vector-icons** | لتوفير أيقونات Material Community Icons الجذابة. |
| **Context API** | لإدارة حالة الثيم (Theme) والبيانات المشتركة في التطبيق. |
| **Animated API** | لإضافة حركات وسلاسة في واجهات المستخدم والرسوم البيانية. |
